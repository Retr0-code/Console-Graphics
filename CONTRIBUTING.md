# Contributing Retr0-code/Console-Graphics

First of all, thank you for making this project better. Before you get started, you should check out [Code of Conduct](https://github.com/Retr0-code/Console-Graphics/blob/main/CODE_OF_CONDUCT.md)

## Code Free Contributions
You can contribute my project without any code experience:
  * You can submit bugs and feature requests
  * Help users with issues
  
## Code contributions
For those who wants to contribute code, they should:
  * Use pull request template
  * Describe in detail additions or fixes
